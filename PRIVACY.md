**SwiftRename**

## Deutsch

### Zusammenfassung

SwiftRename erhebt, speichert oder überträgt keine personenbezogenen Daten. Alle Daten verbleiben ausschließlich auf Ihrem Gerät.

### Datenerhebung

SwiftRename erhebt **keine** personenbezogenen Daten. Die App:

- Sammelt keine Nutzungsdaten oder Statistiken
- Verwendet keine Analyse- oder Tracking-Dienste
- Stellt keine Netzwerkverbindungen her
- Sendet keine Daten an Server oder Dritte
- Enthält keine Werbung

### Lokale Datenspeicherung

SwiftRename speichert folgende Daten ausschließlich lokal auf Ihrem Mac:

- **Einstellungen:** Ihre Konfiguration (Umbenennungsmodus, Dateioperationen, Sprachauswahl) wird in einer lokalen JSON-Datei gespeichert.
- **Ordner-Lesezeichen:** Die von Ihnen ausgewählten Ein- und Ausgabeordner werden als sicherheitsbezogene Lesezeichen gespeichert, damit die App nach einem Neustart weiterhin Zugriff hat.
- **Benutzereinstellungen:** Einfache Einstellungen wie die Sprachauswahl und der Begrüßungsbildschirm-Status werden in den lokalen UserDefaults gespeichert.

Diese Daten verlassen Ihr Gerät nicht und können jederzeit durch Deinstallation der App vollständig entfernt werden.

### Dateizugriff

SwiftRename greift ausschließlich auf Ordner und Dateien zu, die Sie selbst aktiv auswählen (über den Dateidialog oder per Drag & Drop). Die App arbeitet innerhalb der macOS-Sandbox und hat keinen Zugriff auf andere Bereiche Ihres Systems.

### Drittanbieter-Dienste

SwiftRename verwendet keine Drittanbieter-Dienste, SDKs oder Frameworks, die Daten erheben.

### Kinder

Da SwiftRename keine Daten erhebt, bestehen keine besonderen Risiken für Kinder.

### Änderungen

Änderungen an dieser Datenschutzrichtlinie werden in dieser Datei veröffentlicht. Da keine Daten erhoben werden, sind wesentliche Änderungen nicht zu erwarten.

## English

### Summary

SwiftRename does not collect, store, or transmit any personal data. All data remains exclusively on your device.

### Data Collection

SwiftRename collects **no** personal data. The app:

- Does not collect usage data or statistics
- Does not use any analytics or tracking services
- Does not establish network connections
- Does not send data to servers or third parties
- Does not contain advertising

### Local Data Storage

SwiftRename stores the following data exclusively on your Mac:

- **Settings:** Your configuration (rename mode, file operations, language selection) is stored in a local JSON file.
- **Folder bookmarks:** The input and output folders you select are stored as security-scoped bookmarks so the app can retain access after a restart.
- **User preferences:** Simple settings such as language selection and welcome screen status are stored in local UserDefaults.

This data never leaves your device and can be completely removed at any time by uninstalling the app.

### File Access

SwiftRename only accesses folders and files that you actively select (via the file dialog or drag and drop). The app operates within the macOS sandbox and has no access to other areas of your system.

### Third-Party Services

SwiftRename does not use any third-party services, SDKs, or frameworks that collect data.

### Children

Since SwiftRename does not collect any data, there are no specific risks for children.

### Changes

Changes to this privacy policy will be published in this file. Since no data is collected, significant changes are not expected.
